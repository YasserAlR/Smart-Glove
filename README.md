# Smart Glove Project

An assistive wearable device designed to translate hand gestures into real-time audio output for communication assistance.

## 🛠️ Components Used
* Arduino Microcontroller
* Flex Sensors
* Speaker / Audio Module
* Power Supply

## 💡 How it Works
1. Flex sensors detect specific hand gestures.
2. The Arduino processes the analog signals.
3. The system maps the movements to pre-programmed voice commands.
4. Output is played through the speaker in real-time.
